## 为什么选择 MDX？

MDX 允许我在 Markdown 中使用 React 组件，这非常强大。

- 支持所有 Markdown 语法
- 可以导入和使用 React 组件
- 可以使用 JSX 语法

```javascript
// 这是一段代码示例
function hello() {
  console.log('你好，世界！');
}
```
